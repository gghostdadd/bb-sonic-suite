# BB Sonic Suite

Browser-based audio tools built with the Web Audio API. No installs, no plugins — just open and play.

## Tools

### BB Tapedeck v1.0
Reel-to-reel style recorder with variable speed playback, punch-in recording, scrub controls, and WAV export in 16/24/32-bit.

### BB Sampler v1.0
Chromatic sampler with keyboard playback, destructive bit crush and sample rate reduction, interactive trim controls, and 16-voice polyphony.

### BB Edit v1.0
Stereo audio editor with canvas waveform display, click-drag selection, zoom/scroll, cut/copy/paste/delete/crop, fades in/out, normalize, silence, recording, and WAV export.

## Usage

Open any tool directly in your browser. Record from your microphone or load existing audio files. Everything runs locally — no data is sent anywhere.

**Keyboard shortcuts (Sampler):**
- `A S D F G H J K L` — white keys
- `W E T Y U O P` — black keys
- `Z / X` — octave down / up
- `R` — record, `Space` — stop, `Q` — loop, `I` — gate/latch

**Keyboard shortcuts (Tapedeck):**
- `Space` — play / stop
- `R` — record
- `← / →` — rewind / fast-forward
- `↑ / ↓` — speed up / down
- `L` — loop, `E` — erase, `S` — save

**Keyboard shortcuts (Edit):**
- `Space` — play / stop
- `R` — record
- `L` — toggle loop
- `Cmd/Ctrl + X/C/V/Z` — cut / copy / paste / undo
- `Cmd/Ctrl + A` — select all
- `Delete/Backspace` — delete selection
- `+ / -` — zoom in / out
- `0` — zoom fit
- `Escape` — deselect
